This Website!

This Website was created in 2018 to learn some basics of opening up a very simple website, which this certainly is. I started this website on a Digital Ocean Ubuntu LAMP droplet, which filled my needs at the time. Now the website is being run in a Docker container, alongside some other projects. When I learned about Docker, I got it up and running as fast as possible! The website is running nginx and uses PHP. This allows me to more easily modify the website, and run other stuff alongside the website.

The website is completely open source, and is available on my GitHub page (see above). All code is my own -- except for the CSS, which is provided by W3, as well as the icons which are from Font Awesome.

Sticky's Uber Cheats Collection

Sticky's Uber Cheats Collection (SUCC) is an un-finished tool I made to help moderate a gameserver. A lot of people would join the server just to cheat and hack, and we would ban them. Our community required a high burden of proof for hackers, and we kept track of all of the evidence in a big, giant forum thread. The forum thread was hard to search, so when someone would come back off a hacking ban it was difficult to search for them and review old evidence, and to identify and log any alternate accounts. SUCC solved the issue by providing a searchable database where you could freely log information on cheaters, upload videos, write notes, etc. Logins were handled and verified through Steam. SUCC used PHP and MySQL to handle the backend, and looked terible in the front-end, but totally worked.

I never finished this project, but I got through to making basic functionality.
